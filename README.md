## 컬리의 일상
---
## 프로젝트 설명
### 컬리 헤커톤에서 개발했던 컬리 사용자들이 이용하는 커뮤니티 프로젝트입니다.
---
## 프로젝트 목표
### 컬리 사용자들이 컬리 제품과 관련된 컨텐츠를 생산하고 소비함으로써 재소비를 촉진시키는 게 목표입니다.
---
## 프로젝트 설치 및 실행
    # 프로젝트 클론
    git clone https://github.com/hyeon9782/kurly-life.git

    # 패키지 설치
    npm install

    # 프로젝트 실행
    npm run serve
